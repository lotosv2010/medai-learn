# Career Roadmap

AI 工程师职业路线图 · 前端10年老兵的系统学习手册

## 功能特性

- 职业方向决策分析
- AI 应用工程师 vs FDE 对比
- 完整知识体系大纲
- 6 个月学习路线规划
- 知识点进度追踪（本地存储）

## 技术栈

- Next.js 15 (App Router)
- React 19
- TypeScript
- CSS Variables (亮色主题)

## 开发

```bash
# 启动开发服务器
pnpm dev

# 构建
pnpm build

# 启动生产版本
pnpm start
```

## 访问

开发服务器启动后访问 http://localhost:3000
