# /checklist — 知识点掌握清单

用法：`/checklist` 当前阶段 | `/checklist react` 指定模块

## 执行步骤

读取 `docs/learning/progress.json`，结合 $ARGUMENTS 生成互动式知识点检查清单。

**对每个知识点，用以下 4 个级别评估：**

- ⬜ **未接触**：还没学到
- 🟨 **了解**：知道是什么，但讲不清楚
- 🟧 **理解**：能讲清楚原理，但不熟练
- ✅ **掌握**：能手写，能面试讲清楚，能在项目中灵活运用

**输出格式：**

```
## {模块名} 知识点清单

### 核心必掌握（面试高频）
- ✅ 事件循环 & 微任务/宏任务
- 🟧 Promise 手写实现
- 🟨 async/await 底层原理
- ⬜ Generator 与协程

### 进阶理解
...

### 实战应用
...

---
掌握率：{X}/{总数} ({百分比}%)

### 下一步行动
优先攻克：{列出 🟨 和 ⬜ 中最重要的 3 个}
本周目标：完成 {具体知识点}

使用 /quiz {模块} 来测试这些知识点
使用 /explain {知识点} 来深入理解
```

---

**重要**：诚实评估，不要高估。宁可低估促进学习，不要高估产生错误自信。
列出该模块在 medai-learn 项目中有哪些文件/功能对应这些知识点。
